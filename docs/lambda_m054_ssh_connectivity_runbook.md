# Lambda M054 SSH Connectivity Runbook Preview

The M054 runbook preview generated by M053 is non-executable. It may describe a
future supervised milestone only.

The future milestone would require:

1. fresh safety gates and operator approval
2. at most one instance launch
3. provider-visible IP or hostname collection
4. bounded SSH handshake/authentication check only
5. no shell, remote command, file transfer, forwarding, install, or training
6. owned-instance termination
7. termination verification through Lambda read-only discovery/list/get

M053 does not authorize immediate SSH or launch.

## M054A execution package

M054A prepares the reviewer-compatible execution package for a future M054B run.
It still does not launch, SSH, use credentials, or open a network connection.

The M054A package adds:

- a future execution plan for exactly one supervised launch and one SSH
  connectivity/authentication probe
- a private-key reference policy that permits only redacted symbolic references
- a safe OpenSSH command preview with no remote command and no TTY
- static validation for no shell, file transfer, forwarding, install, or training
- an ephemeral one-shot arming artifact
- a reviewer bridge that is the only artifact allowed to expose future one-shot
  request-send/probe permission
- a no-exec audit and non-executable M054B command preview

Standing artifacts must remain `launch_ready=false` and `launch_allowed=false`.
The M054A preview is not a command to run during M054A.
