# Lambda Second Attempt Review

M030 is a review-only milestone after the M029C launch-response-loss incident.
It consumes the closed M029E incident report, M029C run report, current
authorization evidence, and code audits to decide whether a future M031 review
may consider a second launch attempt.

M030 does not launch, terminate, mutate, or spend. A closed M029C incident is
required before the second-attempt review can pass.

Required review outputs:

- second-attempt risk review
- response-loss mitigation review
- correlation plan
- reconciliation plan
- future-M031-only authorization
- go/no-go record

The highest M030 positive state is
`go_for_future_m031_second_launch_review`. It is not launch approval.
