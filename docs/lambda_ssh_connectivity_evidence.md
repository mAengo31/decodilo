# Lambda SSH Connectivity Evidence

Future M054 evidence must prove connectivity-only behavior:

- selected instance ID redacted
- selected instance type and region
- IP or hostname redacted
- SSH client mode
- connection/auth result
- no remote command
- no file transfer
- no port forwarding
- no package install
- no training
- termination report and final discovery
- secret scan

Remote command output and private key material are invalid evidence.
