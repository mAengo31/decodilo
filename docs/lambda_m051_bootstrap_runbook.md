# Lambda M051 Bootstrap Runbook Preview

The M051 runbook preview generated by M050 is non-executable. It describes a
future supervised milestone only.

Default metadata-only flow:

1. Verify M050 authorization artifacts and disabled launch flags.
2. Re-run future pre-launch safety gates.
3. Launch exactly one approved Lambda instance only if all future gates pass.
4. Verify the owned instance through Lambda read-only get/list.
5. Collect provider/local metadata only; do not SSH.
6. Do not install packages, run setup scripts, use cloud-init, or train.
7. Terminate exactly the owned instance in the same supervised run.
8. Verify terminal or absent state through Lambda read-only discovery/list/get.

Any SSH or command execution requires separate explicit operator approval and a
new supervised milestone.

## M052 Closeout

M052 records the successful M051B metadata-only run, reconciles termination, and
attests that no SSH, remote command, package installation, or training occurred.
It is an offline closeout milestone and must not launch, terminate, call live
Lambda read-only APIs, use credentials, or spend money.

The next safe step is a planning-only M053 decision. A successful metadata
bootstrap is not an approval to SSH, run commands, install packages, or train.
